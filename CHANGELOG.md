# Changelog

## v0.1.0

- Initial enterprise HA scaffold.
- Default 3-node RKE2 profile with HAProxy, Keepalived, and Chrony.
- Helm chart for application workloads, webserver provider, Kafka/ZooKeeper, Redis Sentinel scaffold, CloudNativePG, ECK, Logstash, and Zabbix Agent.
- GitHub Actions and GitLab CI pipelines.
- GitHub/GitLab private repository creation scripts.
